### 운영 방식

1. 매주 커리큘럼에 따라 **화요일 18시**까지 공부한 내용을 각자의 블로그(깃허브, 노션, 티스토리 등)에 정리하여 쿠다 노션 페이지에 업로드합니다.
2. 세션은 **나작무 및 랜덤 발제(팀별) → 교육부 세션(github 사용법 등) → 팀별 합동 문제풀이/토의 → 코드 방탈출** 순으로 진행됩니다.
3. 발제 순서는 사전에 정해져 있으며, 각 팀에서 한 명 또는 두 명씩 **사전 과제(노션/개인 블로그 정리 내용)를 바탕으로 발표**를 진행합니다.
4. 발제가 끝나면 교육부가 준비한 Test를 풀고 토의하는 시간을 갖습니다.
5. Test 이후 **코드 방탈출**을 30분간 진행합니다. 주어진 코드의 빈칸을 채우는 방식으로 진행되며, 교재만 참고하여 해결해야 합니다. 먼저 해결한 사람은 퇴실할 수 있습니다.
